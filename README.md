# Ganesha Esthetic - Comando Central

Terminal de control para gestión de turnos y servicios de estética.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 📋 Estructura

- `src/app/` - Next.js App Router
- `src/components/` - Componentes React
- `src/context/` - Estado global (tema, zoom)
- `src/config/` - Design tokens
- `public/` - Archivos estáticos

## ✨ Funcionalidades

- 🌙 Tema oscuro/claro con persistencia
- 🔍 Zoom A+/A-
- 📱 Responsive
- ⚡ Context API para estado global

## 🛠️ Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
